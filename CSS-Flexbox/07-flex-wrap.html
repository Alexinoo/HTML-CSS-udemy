CSS FlexBox - flex-wrap

  • Content/boxes will overflow if the space is not enough to fit all of them
  • flex-wrap : nowrap (default behavior -content will overflow)
  • flex-wrap : wrap (starts a new line in the container if no enough space)
  • flex-wrap : wrap-reverse (starts a new line in the container if no enough space and flips the order in reverse)

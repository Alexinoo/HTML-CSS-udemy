CSS FlexBox - justify-content

  • justify-content (Horizontal alignment)
  • justify-content:flex-start ( aligns the flex items at the start of the container(default) )
  • justify-content:flex-end ( aligns the flex items at the end of the container )
  • justify-content:space-between ( displays the flex items with space between the flex-items)
  • justify-content:space-around (displays the flex-items with space(margins) before, between, and after the lines)
  • justify-content:space-evenly (displays the flex-items with even (margins) before, between, and after the lines)
  • justify-content:center (aligns the flex items at the center of the container)

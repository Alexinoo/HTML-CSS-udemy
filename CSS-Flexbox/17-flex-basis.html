CSS FlexBox - flex-basis:

  • The flex-basis property specifies the initial length of a flexible item.
  • Be aware of padding (Need to set border-sizing-border-box)
  • The flex-basis property takes in a number :
    • A length unit specifying the initial length of the flexible item(s)
    • or a percentage(%) specifying the initial length of the flexible item(s)
  • The flex-basis property is set to auto by default which means that the length is equal to the length of the flexible item
  • Note: If the element is not a flexible item, the flex-basis property has no effect.

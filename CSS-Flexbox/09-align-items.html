CSS FlexBox - align-items

  • align-items (vertical alignment)
  • align-items:stretch ( stretches the flex items to fill the container(default) )
  • align-items:flex-start ( aligns the flex items at the top of the container )
  • align-items:flex-end ( aligns the flex items at the bottom of the container )
  • align-items:center ( aligns the flex items in the middle/center of the container )
  • align-items:baseline ( aligns the flex items such as their baselines aligns - logo example)

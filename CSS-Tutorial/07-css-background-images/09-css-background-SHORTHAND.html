CSS Background SHORTHAND

  • The sequence is background : url() position/size repeat attachment
    • background : url() center/cover no-repeat fixed;
  • We can flip repeat & attachment properties as below :
    • background : url() center/cover fixed no-repeat;
  • However center/cover properties cannot be flipped to cover/center
  • In case you have linear-gradient , the syntax would be like the following
    • background : linear-gradient() , url() center/cover no-repeat fixed;
  • In case you don't have linear-gradient , add the values after url()
    • background : url() center/cover no-repeat fixed;

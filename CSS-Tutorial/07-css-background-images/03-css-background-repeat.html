CSS background-repeat

  • By default, the background-image property repeats an image both horizontally and vertically.
  • We have the following property values for background-repeat
    • background-repeat:repeat - default behavior
    • background-repeat:repeat-x - repeated horizontally
    • background-repeat:repeat-y - repeated vertically
    • background-repeat:space - repeat the image but creates space between
    • background-repeat:round -
    • background-repeat:no-repeat - background image is placed in the same place as the text , However , We need to change the position of the image, so that it does not disturb the text too much.

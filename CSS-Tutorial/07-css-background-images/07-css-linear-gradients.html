CSS linear-gradient()

  • CSS gradients let you display smooth transitions between two or more specified colors.
  • CSS defines three types of gradients: (linear / radial / conic)
  • To create a linear gradient you must define at least two color stops. (Not limited to 2 colors)
  • Color stops are the colors you want to render smooth transitions among
  • You can also set a starting point and a direction along with the gradient effect.
    • Direction - Top to Bottom (this is default) : linear-gradient(to bottom, color-stop1, color-stop2,..);
    • Direction - Bottom to top : linear-gradient(to top, color-stop1, color-stop2,..);
    • Direction - Left to Right : linear-gradient(to right, color-stop1, color-stop2,..);
  • We can also use angles/degrees to have more control instead of predefined directions
    • Equivalent "to top" : linear-gradient( 0deg, color-stop1, color-stop2);
    • Equivalent "to right" : linear-gradient( 90deg, color-stop1, color-stop2);
  • We can also add transparency by using rgba() fxn to define color stops
  • The last parameter in the rgba() function can be a value from 0 to 1, and it defines the transparency of the color:
    • 0 indicates full transparency
    • 1 indicates full color (no transparency).
  • The following example shows a linear gradient that starts from the left. It starts fully transparent, transitioning to full color red:
    • linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1));

CSS background-position

  • The background-position property sets the starting position of a background image.
  • By default, a background-image is placed at the top-left corner of an element, and repeated both vertically and horizontally.
  • There are four different values you can use with this property
    • top -
    • right - places the image on the right
    • left - places the image on the left
    • bottom -
    • center - places the image at the center
  • We can also write the values in percentages(%) in x% y% format
    • background-position: 0 0; - default behavior
    • background-position: 20% 50%; - moves the image 20% horizontally & 50% vertically
    • background-position: 100% 100%; - moves the image 100% horizontally & 100% vertically - right-bottom

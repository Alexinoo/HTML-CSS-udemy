CSS background-size

  • The background-size property specifies the size of the background images.
  • There are four different values you can use with this property
    • background-size : auto; - (default behavior)
    • background-size : cover; - Resize the background image to cover the entire container, even if it has to stretch the image or cut a little bit off one of the edges
    • background-size : contain; - Resize the background image to make sure the image is fully visible - retains the ratio of the image
    • background-size : width height; - (Sets the width and height of the background image.)
    • However with size set to cover - the quality of the image is going to suffer/distorted

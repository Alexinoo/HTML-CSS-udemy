Google fonts

  • If you do not want to use any of the standard fonts in HTML, you can use Google Fonts.
  • Google Fonts are free to use, and have more than 1000 fonts to choose from.
  • We have 2 options of using Google Fonts
    • link option - Just add a special style sheet link in the <head> section and then refer to the font in the CSS.
    • @import option - add import statement into your styles.css (at the top preferrably)
  • Add CSS rules to specify families in the styles.css
    • font-family: 'Permanent Marker', cursive;
    • font-family: 'Roboto', sans-serif;

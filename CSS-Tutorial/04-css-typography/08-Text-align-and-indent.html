text-align and text-indent property

  • text-align - used to align content inside a BLOCK-LEVEL element.
    • left - every content inside a block-level element is aligned to the left by default
    • right - pushes the content inside a block-level element to the right.
    • center - spaces are created on the left and right
    • justify - lines up the content on the left and right edges of the block-level element (the box)
    • inherit
  • text-indent - specifies the indentation of the first line in a text-block

CSS Box Model - Outline/Offset

  • An outline is a line drawn outside the element's border.
  • An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element "stand out".
  • Outline has the following properties:
    • outline-style
    • outline-color
    • outline-width
    • outline-offset
  • The outline-offset property adds space between an outline and the edge/border of an element. The space between an element and its outline is transparent. values can be (-/+)
  • The outline property is a shorthand property for setting the following individual outline properties:
    • outline-width
    • outline-style(required)
    • outline-color

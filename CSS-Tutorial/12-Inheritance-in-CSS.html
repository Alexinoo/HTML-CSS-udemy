Inheritance in CSS

  • Children inherit styles from the parent, unless they have their own styles.
  • In the example below , both <p> and <h2> elements inherit the styles from the parent <div>
  • However, if individual child has it's own styles , they will overwrite the styles from the parent <div>
  • Note : This applies only for matching properties - The child will still inherit other elements from the parent , if it does not specify it's own

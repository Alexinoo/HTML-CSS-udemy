CSS Units - Viewport Units

  • Viewport Units are relative units - relative to the screen(mobile/desktop) etc
  • regardless of whether it is a small / giant screen
  • In web browser terms, a viewport is generally the same as the browser window
    • vh - height -100vh rep 100% of the screen
    • vw - width -100vw rep 100% of the screen
  • allow constructing layouts without any extra hacks

CSS Units - em Units

  • RELATIVE - depends on parent (if no parent - default browser)
  • 1em - 16px of the default browser style
  • 1em - base value (16px) * number(1) = 16px
  • 2em - base value (16px) * number(2) = 32px
  • 2em - base value (10px) * number(2) = 20px
  • settings on browser > Appearance > font size
    • Very small
    • small
    • Medium(Recommended) 16px
    • Large
    • Very large
  • Falls back to 16px(default browser) if no font size is set for the parent

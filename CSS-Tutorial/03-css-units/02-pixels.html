CSS Units - PIXELS

  • pixel is the most basic css unit
  • 1 pixel is one dot on the screen in simple terms
  • pixels are absolute units/values
    • 200px are going to be 200px on both a small or a big screen
  • used with
    • font-size - control the size of the font for our element
    • width - control width of an element
    • height - control height of an element

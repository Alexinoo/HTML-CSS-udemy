Height : auto

  • The height property sets the height of an element.
  • Height is set to auto by default and the element will automatically adjust its height to allow its content to be displayed correctly.
  • Content placed between div.example determines the height of the div
  • If there is more content, the height of the div is going to depend on that content
  • If height is set to a numeric value (e.g px, (r)em,or %) and the content does not fit within the specified height, it will overflow.
  • How the container will handle the overflowing content is defined by the overflow property.(hidden / scroll) values
  • Setting overflow : hidden though - Means that all the extra text that is past 25rem is going to be hidden
  • To see the rest of the text, we can add overflow:scroll which allows us to scroll and see the rest of the content that was hidden

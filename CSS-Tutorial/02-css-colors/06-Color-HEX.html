HEX or Hexadecimal Colors

  • There are four numbering systems : Hexadecimal (Base 16 or hex), decimal (base 10), binary (base 2) and octal (base 10).
  • Hex consists of 16 possible digit values from 0 to 9, followed by six alphabetic characters -- A[10], B[11], C[12], D[13], E[14] and F[15]
  • A Hex color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.
  • In CSS, a color can be specified using a hex value in the form:
  • #rrggbb
    • Where the 2 values range between 00 and ff (same as decimal 0-255).
      • first 2 values (rr) rep red
      • next 2 values (gg) rep green
      • The last 2 values (bb) rep blue
    • e.g, #ff0000 is displayed as red, because red is set to its highest value (ff) and the others are set to the lowest value (00).
      • #000000 - display black
      • #fffff - display white, All values are set ff - same as 255,255,255
      • #00ff00 - display green, The middle values are set to its highest value ff
      • #0000ff - display blue, The last values are set to its highest value ff
  • #rgb (3 Digit HEX Value )- is a shorthand for #rrggbb where :-
    • r, g, and b represent the red, green, and blue components with values between 0 and f.
    • 3-digit hex code are only used when both values (RR, GG, and BB) are the same for each component
    • e.g. #ff00cc, it can be written like : #f0c
    • e.g. #ffcc99, it can be written like : #fc9
    • e.g. #ff00ff, it can be written like : #f0f
    • e.g. #bb5588, it can be written like : #b58

CSS box-shadow property

  • The box-shadow property attaches one or more shadows to an element.
  • The syntax would be like the following
    • box-shadow: h-shadow v-shadow blur-radius color|none|initial|inherit; where
      • h-shadow-length Required. The position of the horizontal shadow (X-axis). Negative values are allowed
      • v-shadow-length Required. The position of the vertical shadow (Y-axis). Negative values are allowed
      • blur-radius - Optional. The blur radius. Default value is 0
      • spread-radius - Optional. The spread radius. A positive value increases the size of the shadow, a negative value decreases the size of the shadow
      • color - Optional. The color of the shadow.

min-width and max-width properties

  • There exist several screen breakpoints:
    • 0px - 576px (S) - Mobile
    • 577px - 768px (M) - Tablet
    • 769px - 992px (L) - Laptop
    • 993px - 1200px (XL) - Desktop
    • 1201px - 1400px (XXL)
  • Max-width :
    • creates a boundary - works as long as we are inside the boundary
    • Essentially : "upto":
  • Min-width :
    • creates a boundary - works if we go ouside the boundary
    • Essentially : "starting from"
  • In summary :
    • max-width sets styles inside the set boundary
    • min-width sets styles outside the set boundary

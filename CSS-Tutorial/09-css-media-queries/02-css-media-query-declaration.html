Media Query Declaration

  • How to declare media queries
    • start with the @media declaration - tells the browser that we have specified a media query
    • set the media type - specify the nature of the device we're working with
      • all - all media type devices
      • print - printers
      • screen - computer screens , tablets , smart-phones
      • speech - screen-raeders that reads tha page out loud
    • We declare the media type after the @media declaration, like this:
      • @media all /screen /print /speech
    • We write the "and" operator if we have two conditions, like this:
      • @media screen and (max-width : 768px){}
    • We don't write the "and" operator if we have one condition
      • @media screen {}
    • You can also skip the media type and work with just min-width & max-width, like this:
      • @media (min-width : 480px) and (max-width : 768px) {}
    • If you have three conditions or more, you can use a comma, like this:
      • @media screen, (min-width : 480px) and (max-width : 768px) {}

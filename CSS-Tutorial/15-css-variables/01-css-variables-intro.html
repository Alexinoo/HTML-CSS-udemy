CSS Variables / Custom Properties Intro

  • CSS variables allows us to store values and re-use it later
  • CSS variables can have a global or local scope.
  • Global variables can be accessed/used through the entire document, while local variables can be used only inside the selector where it is declared.
  • To create a variable with global scope, declare it inside the :root{} selector.
  • To create a variable with local scope, declare it inside the selector that is going to use it.
  • we declare two variables (--blue and --white). Then, we use the var() function to insert the value of the variables later in the style sheet:
  • Advantages of using var() are:
    • makes the code easier to read (more understandable)
    • makes it much easier to change the color values
  • The syntax would be like the following :
    • --varName : value
    • property : var(--varName)

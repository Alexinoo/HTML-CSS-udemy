Transform : scale(width,height)

  • The scale(width,height) method increases or decreases the size of an element (width and height) or (along both X & Y axis)
  • If width and height values are the same we can just have 1 value which represents the 2 sides e.g. scale(0.5)
  • The scaleX(width) method increases or decreases the width of an element. or (along X-axis)
  • The scaleY(height) method increases / decreases the height of an element. or (along Y-axis)
  • units on width/height are not specified

Transform : translate(x-axis,y-axis)

  • The translate property allows you to change the position of elements.
  • The translate property defines x- and y-coordinates of an element in 2D. You can also define the z-coordinate to change position in 3D.
  • Coordinates can be given as only x-coordinates, x- and y-coordinates, or x-, y- and z-coordinates.
    • e.g. translate: 1px 0px -3px;
  • An alternative technique to translate an element is to use CSS transform property with CSS translate() function.
    • e.g. transform: translate(x-axis , y-axis);
  • We also have transform : translateX(value) which is more specific and moves the element along X-axis
  • and transform : translateY(value) which is more specific and moves the element along Y-axis
  • values can be in px or % , however if we use % - this will be relative to the width/height of the parent container

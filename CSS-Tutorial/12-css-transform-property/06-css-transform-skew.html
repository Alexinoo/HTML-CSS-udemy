Transform : skew(deg,deg)

  • The skew(x,y) method skews an element into a given angle where
    • x rep the angles in (degrees) that the element is skewed along X-axis
    • y rep the angles in (degrees) that the element is skewed along Y-axis
  • skewX(deg) - skews an element at a certain angle along X-axis
  • skewY(deg) - skews an element at a certain angle along Y-axis
  • skew(X,Y) - combines the 2
  • skew(deg) - If we provide 1 value , the element is skewed along the X and Y-axises

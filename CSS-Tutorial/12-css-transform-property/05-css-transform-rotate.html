Transform : rotate(deg)

  • The rotate() method rotates an element clockwise or counter-clockwise according to a given degree.
  • Works with both -ve and +ve values
  • Using -ve values will rotate the element counter-clockwise.
  • Not limited to just 360deg - we can even go with 380deg (but creates the same result)
  • Also have rotateZ(deg)
  • We also have rotateX() and rotateY() - very hard to see
  • Most likely the shorthand is used : rotate(deg)

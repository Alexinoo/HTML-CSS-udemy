Font Awesome Icons - Self Hosting - Steps

  • visit Font Awesome Icons
  • Navigate to Docs
  • Navigate to Web
  • Navigate to Host Yourself :
    • click : Host Yourself Web Fonts
    • Navigate and click to download Download the Font Awesome v6 files
  • Reference Font Awesome in Your Project:
    • Link the core fontawesome.css file along with the CSS files for whichever styles you want to use into the <head> of each template or page that you plan to add icons to.
  • Note : Place them before styles.css (preferably)
  • Styling Font awesome icons :
    • Target their classes and style them in the styles.css
    • Add our own classes to already existing i.class and add our own styles

CSS Pseudo-Element ::before

  • The ::before pseudo-element can be used to insert some content before the content of an element.
  • Does not work with image element beause the image itself is treated as the content
  • content : '' property is required even if it is empty
  • If the content is empty , you need to specify some kind of width & height an a background-color to atleast show something
  • The content is added dynamically via CSS
  • use ::before (CSS3) but other uses :before (CSS2 , CSS1)

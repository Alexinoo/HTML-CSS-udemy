Position : fixed

  • An element with position: fixed; is positioned relative to the viewport
  • This means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.
  • A fixed element does not leave a gap in the page where it would normally have been located.
  • browser will use other content to fill up the space

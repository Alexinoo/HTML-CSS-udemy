CSS Display Property - Horizontal Centering

  • Centering Block level Elements :
    • Give it some kind of width (max-width /width) - otherwise it will be spanning all across the width
    • Decide what margin we will give to top/bottom (0+) and set margin left/right to auto
    • margin left/right must be auto !! - can also be written separately
      • margin-left : 0
      • margin-right : 0
  • Centering Inline level Elements :
    • Set text-align to center for the parent element

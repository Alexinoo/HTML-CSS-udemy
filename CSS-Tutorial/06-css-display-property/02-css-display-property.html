CSS Display Property

  • All elements have display property set by default
  • The value might be different but the property will always be there right from the get go
  • The display property specifies the display behavior (the type of rendering box) of an element.
  • Display property values include :
    • Block - Always starts on a new line, and takes up the whole width
    • Inline - Does not start on a new line and spans only content (takes only the width of the content)
  • With CSS , we can control this behavior - we can flip this
    • display inline elements as block elements
    • display block elements as inline elements

Display : inline-block

  • allows us to set a width and height on the element.
    • but with display:inline we cannot
  • the top and bottom margins/paddings are respected
    • but with display: inline they are not
  • does not add a line-break after the element, so the element can sit next to other elements.
    • contrast to with display: block which adds a new line

CSS Grid : fr vs %units

  • fr - Takes into consideration the gap if it's sepcified (CSS does the heavy lifting if the grid-template-rows/columns are specified in fr units)
  • % - Does not take into account of the grid-gap even if it's specified (results to content overflow since the value of the gap is not added/taken into acc behind the scenes)

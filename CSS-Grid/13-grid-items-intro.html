CSS grid items (Child Elements (Items))

  • Grid items refers to items within the CSS grid container.
  • CSS provides numerous properties to work with CSS grid items.
    • grid-column-start & grid-row-start :- determine the location of grid items by specifying where to start the item.
    • grid-column-end & grid-row-end :- determine the location of grid items by specifying where to end the grid item.
    • grid-column :- SHORTHAND property for grid-column-start and grid-column-end properties.
    • grid-row :- SHORTHAND property for grid-row-start and grid-row-end properties.
    • grid-area :-
      • Specifies the grid item's name
      • SHORTHAND property for grid-row-start, grid-column-start, grid-row-end and the grid-column-end properties.
    • justify-self :- Aligns a grid item within a cell along the row/inline axis.
    • align-self :- Aligns a grid item within a cell along the column/block axis.
    • place-self - SHORTHAND property for two properties i.e. align-self and justify-self.
